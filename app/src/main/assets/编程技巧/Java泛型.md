
## 泛型擦除

编译期擦除为 Object

泛型擦除会导致多态冲突，两个方法不同的入参，编译不通过

## 通配符

- 上界通配符（? extends），只能是 ? 或者其子类
- 下界通配符(? super)，只能是 ? 或者其父类

### ? 和 T 的区别？

T 通常表示指定元素只能是 T 类型

而 ? 则表示元素可以是任意类型

？和 T 都表示不确定的类型，区别在于我们可以对 T 进行操作，但是对 ？ 不行，比如如下这种 ：

```
// 可以
T t = operate();

// 不可以
？ car = operate();
```
