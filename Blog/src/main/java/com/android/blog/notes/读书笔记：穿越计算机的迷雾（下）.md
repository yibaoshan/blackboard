### 一、写在前面

在上一篇文章中，我们学习了二进制怎么做加法、一个简单的全加器电路、电报和继电器的发明。

其中，继电器是我们需要关注的重点，计算机能够被创造出来，继电器可以说是最大的功臣

本篇文章主要介绍逻辑门电路，以及各种触发器，目标是做出一个简单的 CPU，大致路线如下：

- 用继电器制作逻辑门（与或非）电路
- 用非门电路制作振荡器
- 用或非门/与非门制作 RS 触发器、D 触发器、上升沿 D 触发器
- 用上升沿 D 触发器制作寄存器
- 制作传输门电路、加法器电路、控制器电路
