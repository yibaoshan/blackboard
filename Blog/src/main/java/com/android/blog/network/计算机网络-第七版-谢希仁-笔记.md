### 1.7.3 具有五层协议的体系结构

OSI的七层协议体系，复杂又不实用

TCP/IP是四层协议体系，分别是：应用层、运输层、网际层、接口层，其中，最下面的接口层没什么具体内容

因此，综合OSI和TCP/IP的有点，采用五层体系结构，即：

- 应用层：端口
- 运输层：tcp/udp
- 网络层：ip地址
- 数据链路层：Mac地址
- 物理层
